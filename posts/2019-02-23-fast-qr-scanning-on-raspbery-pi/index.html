Fast QR Code Scanning on the Raspberry Pi Zero

http://blog.mafrog.info/lelectronique/raspberry-pi/installer-zbar-sur-raspberry-pi/

Download zbar from the official website / sourceforge (don’t use the git repo mirror on github). Make sure you have autoconf installed.

bunzip2 zbar-0.10.tar.bz2
tar -xvf zbar-0.10.tar
mkdir zbar-build
cd zbar-build
../zbar-0.10/configure --prefix=/usr/local --disable-video --without-imagemagick --without-qt --without-gtk --without-python
sudo echo /usr/local/lib > /etc/ld.so.conf.d/zbar.conf
sudo ldconfig

Compiling and installing raspicam is a bit more straightforward.

git clone https://github.com/cedricve/raspicam.git
cd raspicam
mkdir build
cd build
cmake ..
make
sudo make install
sudo ldconfig
